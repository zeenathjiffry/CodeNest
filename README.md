# 🐾 Paws and Whiskers

Paws and Whiskers is a web-based Online Pet Rescue and Rehome Management System designed to help connect rescued animals with loving homes. The platform streamlines the pet adoption process while supporting rescue organizations in managing pet listings and adoption requests efficiently.

## 🚀 Project Overview
This system allows users to browse available rescue pets, view detailed pet profiles, and submit adoption requests. Administrators can manage pets, users, and adoption data through a dedicated admin dashboard.

## ✨ Key Features
- User registration and secure login system  
- Browse and filter available pets for adoption  
- Detailed pet profiles (age, breed, health, description, etc.)  
- Adoption request submission system  
- Admin dashboard for managing pets, users, and admins  
- Add, edit, and delete pet records  
- MySQL database integration  

## 🛠️ Technologies Used
- PHP (MVC Structure)  
- MySQL  
- HTML, CSS, JavaScript  
- Apache / XAMPP / WAMP  

## 🎯 Purpose
To improve the pet adoption process by providing a centralized platform that connects rescued animals with potential adopters efficiently.# CodeNest
