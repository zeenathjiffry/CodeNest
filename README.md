# CodeNest – Android Text Editor App

CodeNest is an Android-based text editor application designed for writing, editing, and testing Kotlin code. It provides essential text editing features along with advanced developer tools such as syntax highlighting, configurable language support, and ADB-based compilation integration.



## Project Overview

The goal of CodeNest is to create a lightweight yet powerful mobile text editor that supports coding on Android devices. It allows users to create, open, save, and edit files while providing a smooth coding experience with syntax highlighting and compiler integration.

The app is primarily focused on Kotlin development but also supports other programming languages through configurable settings.



## Features

### Basic Editor Functions
- Create, open, and save files
- Support for multiple file types (.txt, .kt, .java)
- Copy, cut, paste, undo, redo functionality
- Real-time word and character counting
- Find and replace with case sensitivity and whole-word options

### Syntax Highlighting
- Default Kotlin syntax highlighting (keywords, strings, comments)
- Improved code readability and structure visualization

###  Configurable Highlighting
- Supports JSON/XML-based configuration
- Enables syntax highlighting for other languages like Java and Python

###  ADB Compiler Integration
- Connects to desktop Kotlin compiler via Android Debug Bridge (ADB)
- Sends code for compilation from mobile app
- Displays compilation results and error messages inside the app

###  Error Handling
- Displays compilation errors clearly
- Ensures stable editing and execution workflow



## 🛠️ Technologies Used
- Kotlin (Android Development)
- Android SDK (API 28+)
- JSON / XML (Configuration files)
- Android Debug Bridge (ADB)



##  Purpose

CodeNest demonstrates how a mobile text editor can be extended into a developer tool by combining editing features, syntax highlighting, and remote compilation capabilities in a single Android application.
